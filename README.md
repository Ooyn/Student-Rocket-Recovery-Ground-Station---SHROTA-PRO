# Student-Rocket-Recovery-Ground-Station---SHROTA-PRO
Shrota Pro is a handheld, STM32-based ground station PCB for student rocket recovery. Acting as a relative GPS, it integrates L89 GNSS and a BMM150 compass to pinpoint the rocket's exact landed position. LLCC68 &amp; RYLR998 LoRa modules ensure reliable long-range telemetry reception, guiding users directly to the touchdown site.
