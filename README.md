# Student-Rocket-Recovery-Ground-Station---SHROTA-PRO
Shrota Pro is a custom-engineered, portable ground station PCB specifically designed to track and recover student-built rockets after touchdown. Acting as a specialized, relative GPS system, the handheld device receives critical telemetry data from the rocket and calculates its exact landing coordinates relative to the user's current position, making post-flight retrieval highly efficient.

At the heart of the system is an STM32 microcontroller responsible for real-time processing of incoming telemetry, parsing GPS data, managing sensor inputs, and driving the user interface. The board is powered by a 7.4V dual battery bus, utilizing step-down switching regulators to provide stable 5V and 3.3V rails. This robust power management ensures reliable operation of the MCU, sensitive RF equipment, and the onboard screen interface, allowing users to read relative distances and headings directly in the field. Standardized USB connectivity is also integrated for flashing firmware and serial debugging.

To accurately pinpoint positions, the board utilizes a multi-sensor navigation suite featuring an L89 GNSS module, a NEO-M8N GNSS module, and a BMM150 3-axis geomagnetic sensor that acts as a digital compass to determine the user's precise heading. For reliable long-range telemetry reception, the system incorporates a versatile radio block. It integrates RYLR998 and LLCC68 LoRa modules for robust, low-power communication links, alongside an RFD900x radio modem interface for extended range, ensuring the recovery team maintains a solid, line-of-sight connection to the downed rocket.
